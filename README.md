# Baseball-Projectile-Motion-Simulation
This simulates the motion of a baseball when...
- in a vacuum
- subjected to Newton drag


Various numerical integration methods are used for computing the Newton drag trajectory:
- Euler method
- Tangential method
- 4th order Runge-Kutta method


Additionally, the secant method has been implemented for root finding.
The main program is 'projectile_motion_newton_drag.py'
Additional details on the overview of this program and the theory of projectile motion with and without air drag is found in the attached powerpoint
